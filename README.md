# silo 🚜
A super fast and lightweight TUI based and local only API Client written in Rust

## About
I am passionate about open source and powerful software. I also believe that software should enable and empower others to acheive great things - without trying to taking something from them. That is I am building **silo**. I am tired of all of the api clients trying to force you into signing into their cloud service or jamming AI features where it doesn't belong.

I want silo to be useful, performant software and nothing else.

## AI 
This project will not include any AI features and will not be developed using AI. This project will be 100% human engineered. I beleive that software is a craft, and a way for people to express themselves, I aim to do that with this project while also creating something genuinely useful.
